# Programadores para la Paz

Repositorio de actividades del curso de programación.

- semana1: Ciudadanía digital, información responsable y algoritmia básica
- semana2: Terminal, carpetas, archivos y Git
- semana3: Control de versiones, commits y trazabilidad con Git
- semana4: Node.js, npm, Express y servidor básico
- semana5: Rutas, endpoints, GET, POST y APIs
- semana6: JSON, req.body y peticiones POST
- semana7: Arrays, almacenamiento en memoria y sistema de reportes
- semana8: Linux, terminal, código abierto y seguridad digital
- semana9: Servidores, puertos, logs y estabilidad
- semana10: Metodologías ágiles, backlog, automatización y coordinación comunitaria
- semana11: Base de datos, autenticación de usuarios, interfaz básica y trazabilidad
- semana12: Reto final "Petición en 3 Clics" — integración de backend, base de datos, generación de documentos PDF y automatización con n8n/Telegram
